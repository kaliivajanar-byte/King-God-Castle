# King God Castle Static Site

Этот проект содержит статический сайт `index.html` с `style.css` и изображением.

## Как сделать сайт доступным для всех

### Вариант 1: GitHub Pages
1. Создайте репозиторий на GitHub.
2. Скопируйте сюда все файлы проекта и сделайте коммит.
3. Пушьте в ветку `main`.
4. В настройках репозитория включите GitHub Pages и выберите `Deployment from GitHub Actions`.
5. При каждом пуше на `main` сайт автоматически будет публиковаться.

### Вариант 2: Netlify или Vercel
1. Зарегистрируйтесь на Netlify или Vercel.
2. Подключите репозиторий GitHub.
3. Укажите корневую директорию проекта.
4. Разверните сайт — он получит публичный URL.

### Вариант 3: Просмотр локально
- Просто откройте `index.html` в браузере.
- Или запустите локальный сервер:
  ```bash
  python -m http.server 8000
  ```
  и откройте `http://localhost:8000`.
- Если у вас установлен Python, можно просто запустить `serve.bat` в папке проекта.

## Что уже готово
- `index.html`: главная страница
- `style.css`: стили сайта
- изображение фона должно находиться рядом с `index.html`

## Примечание
Сайт уже статический и полностью работает в браузерах Chrome, Firefox, Edge и других современных браузерах.
